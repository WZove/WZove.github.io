常见命令

目录操作

  1. 创建—==mkdir xxx(创建单级目录)，mkdir -p xx/xx/xx（创建多级目录）==

  2. 删除—==rmdir xxx(删除空目录),rm -rf xxx(删除多级目录)==

文件操作

  1. 创建—==touch xxx.xxx==

  2. 删除—==rm xxx.xxx(带询问的删除)，rm -f xxx.xxx（直接删除）==

  3. 打包—==tar -cvf xxx.tar a.txt b.txt c.txt==

    c:打包文件

    v:显示过程

    f:指定打包后的文件名

  4. 压缩—==tar -zcvf xxx.tar.gz xxx.xxx==

    z:压缩文件

  5. 解压—==tar -xvf xxx.tar.gz(解压到当前目录下)，tar -xvf xxx.tar.gz -C /xxx(解压到指定位置)==

  6. 拷贝—==cp [-adfilprsu] (源文件) （目标位置和文件名）==

    a:相当于pdr

    d:若来源档为连结档的属性(link file)，则复制连结档属性而非文件本身

    f:为强制(force)的意思，若目标文件已经存在且无法开启，则移除后再尝试一次

    i:若目标档(destination)已经存在时，在覆盖时会先询问动作的进行

    l:进行硬式连结(hard link)的连结档创建，而非复制文件本身

    p:连同文件的属性一起复制过去，而非使用默认属性

    r:递归持续复制，用於目录的复制行为

    s:复制成为符号连结档 (symbolic link)，亦即『捷径』文件

    u:若 destination 比 source 旧才升级 destination

  7. 移动—==mv [-fiu] (源文件) (目标位置和文件名)==

    f:force 强制的意思，如果目标文件已经存在，不会询问而直接覆盖

    i:若目标文件 (destination) 已经存在时，就会询问是否覆盖

    u:若目标文件已经存在，且 source 比较新，才会升级 (update)

  8. 查看—==cat xxx.xxx(由第一行开始显示)==

    ​ ==tac xxx.xxx（由最后一行显示）==

    ​ ==nl xxx.xxx(显示行号)==

    ​ ==more xxx.xxx(一页页显示)==

    ​ ==less xxx.xxx(可向前翻页)==

  9. 搜索—==grep xxx [文件名]==

查看信息

  1. 查看进程—==ps -aux==
  2. 管道—==[查询结果作为参数] | grep [查询内容]==

###编辑文件

  1. 创建并编辑—==vi xxx.xxx==

    i—在光标前插入

    a—在光标后插入

    o—在下一行插入

    I—在首行插入

    A—在行末尾插入

    O—在上一行插入

    dd—删除一行

    x—删除一个字符

    ：wq（ZZ）—保存文件

    ：q!—不保存文件

  2. 复制文件内容到另一个文件中—==cat 1.txt >> 2.txt==

  3. 覆盖文件内容—==cat 1.txt > 2.txt==

  4. 权限(除首位，其他为三个一组)例：drwx—— 5 root root 4096 Oct 24 11:31 ./

    d:表示文件

    r:可读；4

    w:可写；2

    x:可执行；1

    第一组：当前用户具有的权限

    第二组：同组用户具有的权限

    第三组：其他人具有的权限

    修改—==chmod u+x 123.txt(表示给123.txt的当前用户加上可执行权限;u：当前用户，g：同组用户，o：其他人)==

    ​ —==chmod u=rwx 123.txt(表示给123.txt的当前用户加上权限)==

    ​ —==chmod u=rwx,g+w,o+r 123.txt==

    ​ —==chmod 777 123.txt==
